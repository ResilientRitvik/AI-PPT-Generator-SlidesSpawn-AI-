# SlideSpawn AI 🚀

SlideSpawn AI is a Streamlit-based AI presentation generator that creates structured slide content from user prompts.

## Features

* AI-generated presentation outlines
* Adjustable slide count
* Multiple presentation categories
* Interactive Streamlit interface

## Technologies Used

* Python
* Streamlit
* OpenRouter API
* GPT Models

## My Contribution

* Built the Streamlit interface
* Integrated AI API for content generation
* Designed workflow for slide creation and user interaction
* Worked on presentation structure and UI organization

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
