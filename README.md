# 🚀 SlidesSpawn AI

AI-powered presentation generator built with Streamlit and OpenRouter.

## Features

* Generate presentations from any topic
* Choose presentation type (College, Business, Marketing, Seminar)
* Select number of slides
* AI-generated content
* Download presentations as PowerPoint (.pptx)
* Modern user interface

## Tech Stack

* Python
* Streamlit
* OpenRouter API
* python-pptx

## Installation

```bash
pip install -r requirements.txt
```

## Run Locally

```bash
python3 -m streamlit run app.py
```
